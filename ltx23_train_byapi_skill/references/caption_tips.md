# Caption 与触发词写法指南

## 什么是 Caption

Caption 是对视频内容的文字描述，训练时每个视频都使用同一个 caption。  
LTX-2.3 使用 Gemma 文本编码器，对自然语言理解较好，caption 越详细越好。

## 好的 Caption 应包含

1. **场景**：在哪里？什么环境？光线如何？
2. **人物**：多少人？大致外形？
3. **动作**：主要动作是什么？动作幅度、节奏？
4. **情绪**：情绪状态、气氛？
5. **声音**（with_audio=true 时重要）：有什么声音？节奏感？

## 触发词（Trigger Word）使用规则

- 触发词会被自动加在 caption 前面（用逗号隔开）
- 例：trigger=`GymKiss`，caption=`在健身房...` → 实际 caption：`GymKiss, 在健身房...`
- 触发词应短、独特、不常见（避免用普通词汇如 "kiss" "dance" 等）
- 触发词推荐用英文 PascalCase 或拼音
- 推理时在 prompt 前加上触发词以激活 LoRA 效果

## 各类别 Caption 参考示例

### 脱外套漏比基尼
```
女性站立，缓缓脱下外套，露出下面的比基尼，动作性感优雅，镜头近距离拍摄。有布料滑落的轻微声音。
```

### 健身房更衣室倒地亲吻
```
在健身房更衣室场景中，男人将女人壁咚储物柜上，两人开始激情亲吻，随后两人在地面上激情亲吻、拥吻，情绪热烈，画面充满张力。有亲吻声和轻微的碰撞声。
```

### 对镜抖臀
```
女性站在镜子前，随着节奏感强的音乐律动，左右摇摆抖动臀部，动作流畅有节奏，背景是简洁的室内环境。音乐节奏明显。
```

### 健身房镜头亲吻
```
在健身房镜子前，男女两人相对而立，缓慢靠近后激情接吻，镜中映出两人的身影，情绪热烈浪漫。有轻微的呼吸声和亲吻声。
```

### 男友衬衫
```
女性穿着宽大的男友衬衫，慵懒地坐在床边或椅子上，画面温馨甜蜜，光线柔和。
```

### 拉出荧幕亲吻
```
电视中的人物在跳舞，电视外的人物将电视中的人物自然地拉出来，人物被拉出来后，两人在电视外开始激情接吻、拥吻，跌跌撞撞，激烈热恋。有舞蹈 BGM 贯穿。
```

### 半蹲亲吻
```
两人面对面，一人半蹲或跪下，仰头望向对方后开始亲吻，动作温柔而浪漫。
```

### 解开衬衫
```
镜头前，人物缓慢解开衬衫纽扣，动作缓慢性感，画面充满张力。有布料摩擦的轻微声音。
```

### 膝枕之吻
```
一人躺在另一人腿上，抬头仰望，两人缓慢靠近后接吻，画面温柔甜蜜，光线柔和。
```

### 爱的小狗加声音（日文）
```
可爱的小狗做出撒娇或依偎的动作，搭配日语配音或字幕，画面温馨可爱。日语配音清晰。
```

## 常见错误

❌ **太短**：`两人接吻` — 没有场景、情绪、声音信息  
❌ **太模糊**：`视频里有人` — 完全没有有效信息  
❌ **触发词混入 caption**：`GymKiss 在健身房接吻` — 触发词会自动加，不要手动重复  
✅ **正确示例**：trigger=`GymKiss`，caption=`在健身房镜子前，男女两人激情接吻，情绪热烈，有呼吸声和亲吻声。`
